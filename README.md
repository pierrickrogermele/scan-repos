SCAN-REPOS
==========

`scan-repos` is a small script utility that prints a brief summary of the state of your Git repositories.

REPOSPATH

INSTALLING
----------

You can use `scan-repos` script as is, or install it using *autoconf*:
``` {.bash}
aclocal
automake --add-missing
autoconf
./configure
make install
```

For uninstalling:
``` {.bash}
make uninstall
```
